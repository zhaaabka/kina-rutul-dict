ʁíxir (4)

Orthography: гъихьир
Part of speech: verb
Glossing label: PV-beat

Meanings

  1. beat — побить
    • hejwanara ʁixir χɨnɨχ / гьейванара гъихьир хыных — ‘Лошадь побила мальчика’
    • χɨnχɨna ʁixir hejwanas / хынхына гъихьир гьейванас — ‘Мальчик побил лошадь’
    • nine χɨnχɨs / rišes ʁixir / нине хынхыс / ришес гъихьир — ‘Мама побила мальчика / девочку’
    • wad halɨs ʁɨlχera? / вад гьалыс гъылхера? — ‘Кого ты бьешь?’
    • χɨnχɨra qʼɨqas ʁixir / хынхыра кьыхъас гъихьир — ‘Мальчик побил телёнка’
    • zirɨra qʼɨqas ʁíxir / зирыра кьыхъас гъи́хьир — ‘Корова побила телёнка’
    • zas χɨnχɨs ʁilxera / зас хынхыс гъилхьера — ‘Я бью мальчика’
    • zas qʼiqas ʁilxera / зас кьихъас гъилхьера — ‘Я бью теленка’

Inflection

Perfective, Gender 3 ʁíxir (4)
Imperfective, Gender 3 ʁilxera (4)
Infinitive, Gender 3 ʁiwxis
Imperative, Gender 3 ʁixʲa (4)
Prohibitive, Gender 3 ʁemelixʲ (4)
