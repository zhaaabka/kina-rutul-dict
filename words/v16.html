eʁéwčʼur

verb
Orthography: егъевчӀур
Glossing label: INSIDE-OUT-move

Meanings

  1. go out
    выйти
    • did χalaː ʁa eʁečʼur / дид халаа гъа егъечӀур
      ‘Отец вышел из дома’
    • eʁelʲčʼur / егъелʼчӀур
      ‘Выходят (люди из дома)’
  2. resettle
    уехать, перехать из какого-либо места
    • ɢina eʁijičʼ mɨχaˤ suqʼʷa / къина егъийичӀ мыхаӀ сукьва
      ‘Уезжай из Кины, поселись в Рутуле’

Inflection

Perfective, Gender 3 eʁéwčʼur
Imperfective, Gender 3 oˤʁówʁura ~ aˤʁábʁura
Infinitive, Gender 3 eʁewčʼus
Imperative, Gender 3 eʁéwčʼeʔ ~ éʁwičʼ
Prohibitive, Gender 3 oˤʁmóˤwuʁ ~ eʁmébčʼ
