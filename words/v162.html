siwígɨr

Orthography: сивигыр
Part of speech: verb
Glossing label: PV-send

Meanings

  1. let go — отпустить
    • χɨnɨχ siga / хыных сига — ‘Мальчика отпусти’
    • zad zer wixʲa ʁaʔ siwigɨra / зад зер вихьʼа гъаъ сивигыра — ‘Я выпустил корову из хлева на улицу’
  2. send — отправить
    • rɨš xijda siriga / рыш хьийда сирига — ‘Девочку за водой отправь’
    • zad šɐxedin paprɨsmɨχta sigira / зад ша̂хьедин папрысмыхта сигира — ‘Я отправил Шахеддина за сигаретами’
  3. filter — процеживать
    • zad nek sifɨja siwigɨra / зад нек сифыйа сивигыра — ‘Я через сито пропустил молоко’

Inflection

Perfective, Gender 3 siwígɨr
Imperfective, Gender 3 siwírgara
Infinitive, Gender 3 siwígas
Imperative, Gender 3 siwíga
Prohibitive, Gender 3 simérwig
