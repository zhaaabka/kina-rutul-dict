ruwʁúr

verb
Orthography: рувгъур
Glossing label: turn

Meanings

  1. go for a walk
    гулять
    • zɨ xɨdɨlnowkan ʁurʁas wɨɣɨr / зы хьыдылновкан гъургъас выгӀыр
      ‘Я гуляю с женой’
    • χɨnɨχ ʁurgata rišekan / хыных гъургата ришекан
      ‘Мальчик гуляет с девочкой’
  2. twiddle
    крутиться
    • tabɨrʁa wuʁurʁara / ruwʁúr / табыргъа вугъургъара / рувгъу́р
      ‘Юла крутится’
    • dünʲe wuʁurʁara werxɨd ʁara huʔ / дуьнʼе вугъургъара верхьыд гъара гьуъ
      ‘Земля вращается вокруг солнца’
  3. search
    искать
    • did ruwʁara χɨnχɨχdɨ / дид рувгъара хынхыхды
      ‘Отец ищет сына’

Inflection

Perfective, Gender 3 ruwʁúr
Imperfective, Gender 3 wuʁúrʁara
Infinitive, Gender 3 wuʁurʁas
Imperative, Gender 3 wuʁúrʁa
Prohibitive, Gender 3 mawɨʁx
