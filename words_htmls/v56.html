keʁéwčʼur

verb
Orthography: кегъевчӀур
Glossing label: SURF-OUT-move

Meanings

  1. quit (habit)
    бросать (о дурной привычке)
    • pʼapʼrɨsakla keʁijičʼ! / пӀапӀрысакла кегъийичӀ!
      ‘Брось курить!’
    • ʁarɨm mɨklʲa wɨ keʁijičʼ! / гъарым мыклʼа вы кегъийичӀ!
      ‘Бросай пить!’
  2. wade in
    перейти вброд
    • lecʼura tiʔ zer keʁewčʼur / лецӀура тиъ зер кегъевчӀур
      ‘Корова прошла через реку’
    • lecʼura keʁwičʼ! / лецӀура кегъвичӀ!
      ‘[Корове:] пройди через реку!’
    • lecʼura keʁijičʼ! / лецӀура кегъийичӀ!
      ‘[Человеку:] пройди через реку!’
  3. separate (from group, herd)
    отделиться (от группы, стада)
    • sa qʼabal süriekla ʁaʔ keʁwij / са кьабал суьриекла гъаъ кегъвий
      ‘Одного барашка со стада отдели’
    • cʼɨ sürija keʁewčur abxɨr / цӀы суьрийа кегъевчур абхьыр
      ‘От отары отделилась коза’

Inflection

Perfective, Gender 3 keʁéwčʼur
Imperfective, Gender 3 keʁébčʼura
Infinitive, Gender 3 keʁewčʼus
Imperative, Gender 3 keʁwičʼ
Prohibitive, Gender 3 keʁméwčʼ
